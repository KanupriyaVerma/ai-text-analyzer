# 🤖 AI Text Analyzer

A full-stack web app that analyzes text and provides:

- Summary
- Sentiment
- Suggestions
- Word Count

## Features
- Chat-style UI
- Voice input
- Sentiment colors
- Input validation

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript

## Run

pip install flask flask-cors  
python app.py  

Open index.html in browser
